## 🏆 КЛЮЧЕВЫЕ ОТКРЫТИЯ

### ✅ **УНИВЕРСАЛЬНЫЙ ПАТТЕРН РЕШЕНИЯ:**

**Все современные дизайн-системы решают нашу проблему одинаково:**

1. **СЕМАНТИЧЕСКИЕ РОЛИ** → `--text-secondary` вместо `#666666`
2. **КОНТРАСТНЫЕ ПАРЫ** → `--surface + --on-surface` автоматически
3. **ФУНКЦИОНАЛЬНАЯ ИЕРАРХИЯ** → каждый элемент знает свою роль
4. **АВТОМАТИЧЕСКАЯ АДАПТАЦИЯ** → новые темы работают без настройки

### ✅ **КОНКРЕТНЫЕ РЕШЕНИЯ НАШИХ ПРОБЛЕМ:**

**ПРОБЛЕМА 1: Стрелка → "почти нигде не видна"**  
**РЕШЕНИЕ:** `color: var(--notification-text-secondary)` - семантическая роль

**ПРОБЛЕМА 2: Статус "прочитано" только в 1 теме из 5**  
**РЕШЕНИЕ:** `var(--notification-status-read-bg) + var(--notification-status-read-text)` - контрастная пара

**ПРОБЛЕМА 3: Белые подложки таблиц в темных темах**  
**РЕШЕНИЕ:** `var(--notification-surface)` - семантическая поверхность
