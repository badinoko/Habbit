## 🎨 ДИЗАЙН ТРЕБОВАНИЯ (ИЗ ТВОИХ ПРИМЕРОВ):

### **СТИЛИСТИКА:**
- Glassmorphism: прозрачные элементы с размытием
- Многослойные тени для визуальной глубины
- Плавные переходы: `transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1)`
- Скругленные углы: `border-radius: clamp(12px, 2vw, 24px)`

### **АДАПТИВНОСТЬ:**
- CSS Grid для сеток карточек
- `clamp()` для адаптивных размеров
- Mobile-first подход
- Breakpoints: 768px, 1200px, 1920px

### **ЦВЕТА (ЧЕРЕЗ ПЕРЕМЕННЫЕ):**
- Фоны: `var(--surface-1)`, `var(--surface-2)`, `var(--surface-3)`
- Текст: `var(--text-primary)`, `var(--text-secondary)`
- Акценты: `var(--primary-color)`, `var(--accent-color)`
- Границы: `var(--border-color)`
