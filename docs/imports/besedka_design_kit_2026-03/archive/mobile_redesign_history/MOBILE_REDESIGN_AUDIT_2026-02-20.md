# Мобильная адаптация Беседки - Критические проблемы

> **Дата:** 2026-02-20
> **Устройство:** Xiaomi 12 PRO
> **Скриншоты:** `docs/mobile_screenshots/`

---

## 🚨 КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. Главная страница - катастрофа
**Скриншот:** photo_4
- Огромные пустые поля по бокам
- Контент занимает ~60% ширины экрана
- Потеря полезной площади

### 2. Кнопки не вмещают свои заголовки
**Скриншоты:** photo_1, photo_2, photo_3
- Текст обрезается
- Иконки и текст накладываются
- Нет адаптации размера шрифта

### 3. Три разных алгоритма расположения фильтров
**Проблема:** В News, GrowReports, Gallery - верхние панели расположены по разным принципам

**News (photo_1, photo_7):**
- Фильтры: 📰 🕐 🔥 💬 + 🔍 🎯 ✅
- Переключатель вида: справа вверху (сетка/список)
- Относительно нормально

**GrowReports (photo_2, photo_8):**
- Верхняя строка: 👤 ➕ [сетка] [список]
- Нижняя строка: 🌱 🕐 🔥 💬 🌿 ✅ + 🔍 🎯 ✅
- Две строки кнопок - слишком громоздко

**Gallery (photo_3):**
- Верхняя строка: 🖼️ 🔒 ➕ [сетка] [список]
- Нижняя строка: 📷 🕐 🔥 💬
- Пустое пространство в нижней строке
- Неоптимальное использование места

### 4. Списочный вид почти не отличается от плиточного

**News (photo_7 - списочный вид):**
- Превью всё ещё огромное (занимает ~40% карточки)
- Метаданных мало
- Заголовок обрезается "Техники тр..."
- **РЕШЕНИЕ:** Радикально уменьшить превью, больше места метаданным

**GrowReports (photo_8 - списочный вид):**
- То же самое - превью слишком большое
- Название: "Второй тест..."
- Мало информации о репорте
- **РЕШЕНИЕ:** Превью 80x80px, остальное - текст

**Gallery:**
- В галерее вообще почти нет разницы между видами
- Оба вида показывают большие изображения
- **РЕШЕНИЕ:** Списочный вид = маленькое превью + название + описание + дата

**Что НЕ работает:**

- Заголовки обрезаются
- Пустые поля по бокам

### GrowReports (Гроу-репорты)
**Что НЕ работает:**
- Две строки кнопок управления (громоздко)
- Списочный вид (photo_8) - превью огромное
- Мало информации о сортах/авторе

### Gallery (Галерея)
**Что НЕ работает:**

- В списочном виде должно быть маленькое превью + текст
- Пустое пространство в панели фильтров

### Homepage (Главная)
**Что НЕ работает:**
- Огромные пустые поля по бокам
- Контент сжат к центру
- Неэффективное использование экрана

### Детальные страницы
**Gallery detail (photo_5, photo_6):**
- Комментарии работают нормально
- Реакции видны
- Форма комментирования адекватная

## 💡 РЕШЕНИЯ

### Главная страница
```css
/* БЫЛО: контент сжат к центру */
.container {
    max-width: 600px; /* слишком узко! */
}

/* НАДО: контент на всю ширину с разумными отступами */
@media (max-width: 768px) {
    .container {
        max-width: 100%;
        padding: 0 16px;
    }
}
```

### Списочный вид - превью 80x80px
```css
/* News/Gallery/GrowReports - списочный вид */
.list-view .card-image {
    width: 80px;
    height: 80px;
    flex-shrink: 0;
}

.list-view .card-content {
    flex: 1;
    padding-left: 12px;
}
```

### Унификация панелей управления
```html
<!-- ЕДИНЫЙ паттерн для всех разделов -->
<div class="content-controls">
    <!-- Строка 1: основные действия -->
    <div class="controls-row-main">
        <button class="btn-user-filter">👤</button>
        <button class="btn-add">➕</button>
        <div class="view-switcher">
            <button class="btn-grid">⊞</button>
            <button class="btn-list">☰</button>
        </div>
    </div>

    <!-- Строка 2: фильтры (свернуть на мобиле?) -->
    <div class="controls-row-filters">
        <button class="filter-all active">📰</button>
        <button class="filter-recent">🕐</button>
        <button class="filter-hot">🔥</button>
        <button class="filter-comments">💬</button>
        <!-- Остальное в dropdown -->
        <button class="btn-search">🔍</button>
        <button class="btn-filter">🎯</button>
        <button class="btn-mark-read">✅</button>
    </div>
</div>
```

### Адаптивные кнопки navbar
```css
@media (max-width: 480px) {
    .nav-link span {
        font-size: 0.75rem; /* меньше шрифт */
    }

    .nav-link {
        padding: 0.5rem 0.75rem; /* меньше отступы */
    }
}
```
