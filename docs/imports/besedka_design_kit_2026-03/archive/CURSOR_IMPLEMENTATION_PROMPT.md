### 2. Использовать ТОЛЬКО SSOT компоненты!

**НЕ ПИСАТЬ новый код для:**
- Карточек → `unified_card.html`
- Фильтров → `unified_filters.html`
- Комментариев → `unified_comment_section.html`
- Реакций → `unified_reactions.html`
- Drawer фильтрации → уже существует!
- **Модальных окон → `unified_modal.html`** ⭐ Использовать для подтверждения "Прочитать все"

### 3. Использовать CSS переменные!

```css
/* ✅ ПРАВИЛЬНО */
color: var(--bs-text-primary);
background: var(--bs-card-bg);
border: 1px solid var(--bs-border-color);

/* ❌ НЕПРАВИЛЬНО - hardcoded цвета! */
color: #333333;
background: #ffffff;
```

### 4. Система 5 тем

| Тема | Ключевые особенности |
|------|----------------------|
| Light | Светлый фон, тёмный текст |
| Dark | Тёмный фон #1a1a2e, светлый текст |
| Cupcake | Розовый/кремовый |
| Emerald | Зелёный/изумрудный |
| Synthwave | Фиолетовый/неон |

Переключение: `BesedkaMasterThemes.apply('dark');`

### 5. Glassmorphism паттерн

```css
.glass-card {
  background: var(--glass-bg);
  backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 16px;
}
```

### Этап 1: Главная страница новостей (`/news/`)

**Прототип:** `news-prototype-v2-final.tsx` ✅ ПРИНЯТ

**Структура Filter Bar:**
```
┌────────────────────────────────────────────────────────────────┐
│ 📰 Новости                                        [🔲 ☰]      │  ← Header + переключатель видов
├────────────────────────────────────────────────────────────────┤
│ [📰 Все] [🕐 Новые] [🔥 Попул.] [💬 Обсужд.]  [🔍] | [🔍] | [✅]│  ← Filter bar
├────────────────────────────────────────────────────────────────┤
│ 🔍 [Поиск по новостям...                              ] [✕]   │  ← Строка поиска (скрываемая)
└────────────────────────────────────────────────────────────────┘
```

**Новые фичи:**
1. ☐ Переключатель видов в HEADER (сетка/список)
2. ☐ Кнопки сортировки: Все | Новые | Популярные | Обсуждаемые
3. ☐ **Поисковая строка** (скрываемая, появляется по клику на 🔍)
4. ☐ Кнопка "Фильтры" с счётчиком активных + крестик сброса рядом
5. ☐ Кнопка "Прочитать все" → **с модальным окном подтверждения!**
6. ☐ Индикатор прочитанности (зелёная точка + opacity)
7. ☐ Счётчики на карточке: 👁 просмотры, 💬 комментарии
8. ☐ Badge источника: 📰 Беседка / 🔗 Внешний

**ВАЖНО: Модальное окно подтверждения**
При нажатии "Прочитать все" → показать SSOT модалку `unified_modal.html`:
- Заголовок: "Отметить все как прочитанные?"
- Текст: "N непрочитанных новостей будут отмечены"
- Кнопки: "Отмена" / "Да, отметить"

**ВАЖНО: Суммирование фильтров**
Поиск работает ВНУТРИ отфильтрованных результатов:
- Выбрана категория + введён поиск = ищем в этой категории
- Выбрана сортировка + поиск = ищем среди отсортированных

**Категории (6 штук включая "Все"):**
| ID | Название | Иконка | Цвет |
|----|----------|--------|------|
| all | Все | 📰 | #6B7280 |
| general | Общие новости | 📢 | #EF4444 |
| cultivation | Культивация | 🌱 | #10B981 |
| genetics | Генетика | 🧬 | #8B5CF6 |
| technology | Технологии | ⚡ | #3B82F6 |
| research | Исследования | 🔬 | #F59E0B |

**Сортировка (4 кнопки):**
| ID | Название | Иконка |
|----|----------|--------|
| all | Все | 📰 |
| new | Новые | 🕐 |
| popular | Популярные | 🔥 |
| discussed | Обсуждаемые | 💬 |

**Breakpoints для Filter Bar (ФИНАЛЬНЫЕ!):**
| Ширина | Поведение |
|--------|-----------|
| ≥850px | Полные кнопки с надписями: [📰 Все] [🕐 Новые] ... |
| 550-849px | Только значки полного размера, разделители между правыми кнопками |
| 400-549px | compactMode: уменьшенные кнопки (p-2), минимальные gaps, без разделителей |

**⚠️ ВАЖНО: Система реакций**

**SSOT компонент:** `templates/includes/partials/unified_reactions.html`

**Логика показа:**
- Показываем ВСЕ счётчики которые есть (flex-wrap на 2 строки если надо)
- Счётчики КЛИКАБЕЛЬНЫ — по клику ставим/снимаем свою реакцию
- Кнопка "+" появляется ТОЛЬКО если есть неиспользованные реакции
- Пикер показывает ТОЛЬКО неиспользованные реакции
- Если все 10 использованы → кнопки "+" нет вообще!

**Ограничения (backend):**
- Пользователь может поставить только ОДНУ реакцию на пост
- Автор НЕ может реагировать на свой контент (проверка уже есть!)
- AJAX endpoint: `/news/ajax/toggle-reaction/`

**Позиционирование пикера:**
- Пикер появляется НАД кнопкой "+"
- Правая граница пикера = правая граница кнопки "+" (right: 0 относительно контейнера кнопки)
- Кнопка "+" и пикер обёрнуты в `position: relative` контейнер

**Стилизация пикера (glassmorphism):**
```css
.bs-reactions-picker {
    position: absolute;
    bottom: 100%;
    right: 0;
    margin-bottom: 8px;
    padding: 6px;
    background: rgba(30, 30, 46, 0.75);
    backdrop-filter: blur(16px);
    border: 1px solid rgba(255, 255, 255, 0.15);
    border-radius: 12px;
    box-shadow: 0 8px 32px rgba(0, 0, 0, 0.4);
}

.bs-reactions-picker button {
    width: 32px;
    height: 32px;
    font-size: 18px;
    border-radius: 8px;
}
```

**Закрытие пикера:** по клику вне пикера (невидимый fixed overlay)

**Sticky header:**
- Прозрачность + blur как у navbar
- Бесшовный переход с navbar (те же параметры glassmorphism)
- Кнопки: Назад | Поделиться | В закладки

**Sidebar sticky без дёрганья:**
```css
.sidebar {
    position: sticky;
    top: 96px; /* top-24 в Tailwind = 6rem = 96px */
}
```

**Sticky header (glassmorphism как у navbar):**
```css
.news-detail-header {
    position: sticky;
    top: 0;
    z-index: 20;
    background: rgba(10, 10, 20, 0.85);
    backdrop-filter: blur(16px);
    border-bottom: 1px solid rgba(255, 255, 255, 0.1);
}
```
ВАЖНО: Параметры прозрачности и blur должны совпадать с navbar для бесшовного перехода!
