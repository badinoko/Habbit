# DIARY Detail Page — User Flows

## ВАЖНОЕ ОТКРЫТИЕ: РЕАЛЬНАЯ АРХИТЕКТУРА

**Обнаружено на 2025-08-14:**
Дневник — это НЕ модальное окно, а **обычная прокручиваемая страница** с полной структурой.

### 1. Основная навигация по дневнику

#### 1.1 Прокрутка по неделям
- **Вертикальная прокрутка**: Недели расположены вертикально, одна за другой
- **Плавная навигация**: Прокрутка от Week 1 до Week 13
- **Хронологический порядок**: Недели идут по возрастанию
- **Infinite scroll**: Динамическая подгрузка контента

#### 1.2 Timeline навигация (внизу страницы)
- **Позиция**: Fixed внизу экрана
- **Структура**: Таббы с номерами недель
- **Цветовое кодирование**:
  - **ВЕГА**: Недели 1-8 (вегетативная фаза)
  - **ЦВЕТ**: Недели 9-13 (фаза цветения)
- **Клик по табу**: Прокрутка к соответствующей неделе
- **Активный таб**: Подсвечивается текущая видимая неделя

### 2. Структура каждой недели

#### 2.1 Заголовок недели
- **Номер**: "Week X"
- **Фаза**: "Вегетация" или "Цветение"
- **Timestamp**: "3mo ago", "2mo ago", "8d ago"

#### 2.2 Фото галерея
- **Layout**: Горизонтальная сетка
- **Количество**: 3-7 фотографий на неделю
- **Размеры**: Адаптивные под контейнер
- **Interaction**: Клик для увеличения (lightbox)

#### 2.3 Метрики (ключевой элемент)
- **Layout**: Сетка кликабельных карточек
- **Иконки**: Каждая метрика с соответствующей иконкой
- **Значения**: Числовые данные + единицы измерения
- **Типы метрик**:
  - Height (рост растения)
  - Light Schedule (режим освещения)
  - Day/Night Air Temp (температуры)
  - pH, Air Humidity
  - Pot Size, Watering Volume
  - Lamp Distance
  - Solution/Substrate Temp

#### 2.4 Nutrients секция
- **Раскрывающаяся**: "Nutrients [количество]"
- **Структура**: Картинка + название + дозировка
- **Примеры**: "Basis A - 2 mll", "Mills C4 - 1.5 mll"
- **Брендинг**: Mills Nutrients с соответствующими иконками

#### 2.5 Описание недели
- **Автор**: Ссылка на профиль гровера
- **Текст**: Заметки и наблюдения на английском
- **Стиль**: Персональный, неформальный

#### 2.6 Interactions
- **Лайки**: Счетчик + кнопка лайка
- **Комментарии**: Ссылка на секцию комментариев
- **Share**: Кнопка поделиться
- **Перевод**: Автоматический перевод описания

#### 2.7 Used Techniques
- **Техники**: LST, Топпинг, Дефолиация, HST, Transplantation
- **Визуализация**: Тэги с названиями техник
- **Связь с неделями**: Показывает, что именно делалось

### 3. Equipment Bar (верхняя секция)

#### 3.1 Горизонтальная витрина оборудования
- **Seeds**: Anesia Seeds "Future #1®"
- **Nutrients**: Mills Nutrients
- **Lighting ВЕГА**: Mars Hydro TS3000 420W LED
- **Lighting ЦВЕТ**: Spider Farmer SF-4000
- **Tent**: Mammoth Pro+ 120
- **Fan**: Prima Klima PK125
- **Filter**: Prima Klima VK125
- **Substrate**: BioBizz Light-Mix
- **Environment**: "В помещении"

#### 3.2 Summary техник
- **Дефолиация**: недели 4-8, 8-13
- **Transplantation**: недели 2, 4, 6, 9-10
- **LST**: недели 3-8, 8-13
- **HST**: недели 7, 11, 13
- **Топпинг**: недели 3-8, 8-11, 13

### 4. Comments System

#### 4.1 Форма комментариев
- **Input**: "Выскажи свое мнение"
- **File upload**: Возможность загрузки файлов
- **Login required**: Требуется авторизация

#### 4.2 Комментарии thread
- **Структура**: Аватар + имя + timestamp + текст
- **Interactions**: Лайки, Перевод, Ответить
- **Sorting**: По популярности
- **Count**: "7 комментария"

#### 4.3 Thread navigation
- **Replies**: Вложенные ответы с отступами
- **Mentions**: @username ссылки
- **Timeline links**: Ссылки на конкретные недели

### 5. Mobile adaptation

#### 5.1 Responsive behavior
- **Timeline tabs**: Горизонтальная прокрутка на мобильных
- **Photo galleries**: Адаптивная сетка
- **Metrics cards**: Переход в вертикальный layout
- **Equipment bar**: Вертикальный стэк или карусель

### 6. Key UI Patterns

#### 6.1 Навигационные элементы
- **Breadcrumbs**: Возврат к списку дневников
- **Timeline**: Fixed навигация по неделям
- **Follow buttons**: Follow Author / Follow Diary

#### 6.2 Content organization
- **Chronological flow**: От Week 1 к Week 13
- **Phase separation**: ВЕГА → ЦВЕТ переход
- **Equipment context**: Постоянная привязка к оборудованию

#### 6.3 Social features
- **Stats**: Followers, Likes, Comments, Views
- **Community**: Comments thread
- **Sharing**: Follow, Share, Like actions

## КРИТИЧЕСКОЕ ПОНИМАНИЕ ДЛЯ РЕДИЗАЙНА:

1. **Это полноценная страница**, а не модальное окно
2. **Вертикальная прокрутка** — основной способ навигации
3. **Timeline внизу** — дополнительная быстрая навигация
4. **Каждая неделя** — самостоятельный блок с полной информацией
5. **Метрики** — ключевой элемент для отслеживания прогресса
6. **Social aspect** — комментарии и взаимодействие с сообществом

## Структурная карта страницы (сверху вниз)

Источник: длинный скрин `docs/growreports/reference/GROWREPORTS_REFERENCE_SPECS/diary_detail/screens/docs-GROWREPORTS-REFERENCE-SPECS-detail-diary-screens-1440-fullpage.png`.

1) Header дневника
   - Заголовок дневника (имя сорта/диария)
   - Автор: аватар, имя, бейджи, дата последнего обновления
   - Краткая панель действий/статистик: Followers, Likes, Comments, Views, «Follow»

2) Equipment/Context Bar (горизонтальная витрина)
   - Плитки с иконками: Strain/Seeds, Fertilizers, Lights (Veg/Flower), Tent, Substrate, Environment, Pots, Watering, CO2 и т.д.
   - Доп. полоса-резюме техник (LST, Дефолиация, HST, Transplantation) с указанием недель

3) Summary блока прогресса роста
   - Полоса «Spent N days» с распределением фаз: Ger/Veг/Flo/Har
   - Справа — иконки «Energy/Taste» и краткие показатели (урожай, площадь, сложность и пр.)

4) Графики/Метрики высокого уровня (если присутствуют на конкретном дневнике)
   - Пример: «Height (cm) vs Weeks» — линейный график
   - Переключатели: Height, Day/Night Temp, Air Humidity, PPM, PH, Light schedule, Solution temp, Pot size, Lamp distance

5) Timeline — недельные блоки (повторяющаяся секция на всю длину страницы)
   - Header недели: «Week X · Фаза» + «Nmo ago»
   - Галерея недели: 3–7 фото (кликабельные, открывают lightbox)
   - Сетка карточек метрик недели:
     - Высота растения, график света, Темп. день/ночь, pH, Влажность, PPM/TDS, Temp раствора/субстрата, Объем полива, Размер горшка, Дистанция до лампы
   - Nutrients (раскрывающийся список): логотип, название, дозировка, единицы
   - Notes: автор, текст заметок, Translate
   - Interactions: Like, Комментарии, Share
   - Used Techniques (тэги): LST, HST, Топпинг, Дефолиация и т.д.

6) Комментарии (общая секция ближе к низу страницы)
   - Форма добавления комментария (требует логин), загрузка файлов
   - Тред с ответами: аватар, имя, время, текст, лайки, перевод, «ответить»

7) Sticky нижняя навигация по неделям
   - Горизонтальная лента «G/ВЕГА/ЦВЕТ/ХАРВ» с номерами недель (1…13)
   - Клик скроллит к соответствующему недельному блоку; активная неделя подсвечена

Примечание о модальности: базовая детальная страница дневника — полноценная вертикально прокручиваемая страница. Отдельно у GrowDiaries присутствуют модальные окна (например, просмотр фотографий и некоторые недельные попапы), см. `../detail/flows/week_modal.md`. В данном документе описана именно основная детальная страница.
