# LIST Page — User Flows

## Главная страница (сетка дневников)

### F1: Поиск дневников
1. **Начальное состояние**: пользователь на главной странице
2. **Действие**: клик в поле поиска textbox[ref=e118]
3. **Результат**: активация поля ввода, показ плейсхолдера "Search among XXXXX diaries"
4. **Ввод запроса**: типизация поискового запроса
5. **Действие**: клик кнопки поиска или Enter
6. **Результат**: фильтрация карточек по запросу, обновление счетчика

### F2: Фильтрация по тегам
1. **Начальное состояние**: полный список дневников
2. **Скролл фильтров**: горизонтальная прокрутка тегов (Все, В помещении, Аутдор, etc.)
3. **Клик по тегу**: выбор фильтра (LED, LST, СОГ, СКРОГ, etc.)
4. **Состояние**: активный тег выделяется
5. **Результат**: мгновенная фильтрация карточек, анимация исчезновения/появления

### F3: Навигация к дневнику  
1. **Ховер карточки**: показ дополнительной информации
2. **Клик по карточке**: переход к detail странице дневника
3. **Breadcrumb**: сохранение навигационного пути
4. **Back navigation**: возврат к filtered состоянию списка

### F4: Sidebar интеракции
1. **Strains раздел**: клик по сорту → страница сорта с характеристиками
2. **Shorts раздел**: клик по видео → воспроизведение в lightbox/модалке
3. **Recent Harvests**: клик → переход к завершенным дневникам
4. **Featured Brands**: карусель брендов, клик → страница бренда

### F5: Infinite scroll / Pagination
1. **Скролл вниз**: подгрузка следующих карточек
2. **Индикатор загрузки**: спиннер во время fetch данных
3. **Буферизация**: предзагрузка изображений следующих карточек
4. **Состояние**: управление количеством загруженных элементов
