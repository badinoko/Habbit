# DETAIL Page — User Flows  

## Страница дневника (timeline недель)

### F1: Timeline навигация
1. **Начальное состояние**: страница открывается с восстановлением scroll позиции
2. **Scroll по неделям**: вертикальная прокрутка через week cards
3. **Week anchors**: быстрые переходы к конкретным неделям (Week 1, Week 2, etc.)
4. **Scrollspy**: активация текущей недели в side navigation (если есть)
5. **Progressive loading**: подгрузка контента недель при приближении к viewport

### F2: Галерея изображений
1. **Hover превью**: увеличение изображения при наведении
2. **Клик по изображению**: открытие в lightbox/fullscreen режиме  
3. **Lightbox навигация**: листание между фото недели (prev/next)
4. **Zoom функция**: приближение для детального просмотра
5. **Закрытие**: ESC/клик вне изображения для закрытия lightbox

### F3: Интеракции с контентом
1. **Лайки**: клик по heart icon → добавление/удаление лайка с анимацией
2. **Комментарии**: раскрытие секции комментариев под неделей
3. **Follow автора**: клик на "Follow" → подписка с обновлением счетчика  
4. **Share**: шеринг конкретной недели через соцсети/URL

### F4: Equipment/Specs раздел
1. **Hover сиды**: показ детальной информации о семенах
2. **Клик по бренду**: переход на страницу производителя (Pyramid Seeds)
3. **Lights/Nutrients**: раскрытие подробных характеристик оборудования
4. **Compare**: сравнение используемого оборудования с другими дневниками

### F5: Hero секция интеракции
1. **Stats наведение**: детализация метрик (36 Likes → список пользователей)  
2. **Author клик**: переход к profile страницы автора
3. **Breadcrumb**: навигация обратно к списку с сохранением фильтров
4. **Badge клик**: ("Одобрен Seedstockers") → информация о программе одобрения

### F6: Mobile адаптация
1. **Touch gestures**: swipe между неделями на мобильных
2. **Pinch zoom**: увеличение изображений жестами
3. **Sticky header**: фиксация заголовка при скролле
4. **Responsive breakpoints**: адаптация layout 320px → 1920px
