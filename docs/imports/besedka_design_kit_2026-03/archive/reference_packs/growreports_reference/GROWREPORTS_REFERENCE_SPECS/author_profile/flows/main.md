# AUTHOR PROFILE Page — User Flows

## Страница профиля автора (MindFlowers68 example)

### F1: Profile интеракции  
1. **Avatar hover**: показ full-size изображения
2. **Message клик**: открытие диалога/формы сообщения
3. **Follow клик**: подписка с анимацией обновления счетчика (80 → 81)
4. **Stats клик**: переход к детальным рейтингам/позициям

### F2: Navigation Tabs
1. **Info tab**: показ базовой информации (активна по умолчанию)
2. **Diaries tab**: сетка дневников автора (34 штуки)
3. **Reviews tab**: отзывы на сорта с рейтингами (400+)
4. **Подписан tab**: список людей, на которых подписан автор
5. **Followers tab**: список подписчиков автора
6. **Lazy loading**: подгрузка контента табов по требованию

### F3: Stats Grid интеракции
1. **Global Position клик**: переход к полным рейтингам (#230 Global pos.)
2. **Harvests клик**: фильтрация дневников по завершенным (17 Harvests)
3. **Experience hover**: детализация опыта (2 года)
4. **Answers клик**: переход к вопросам с ответами автора (65 Answers)

### F4: Preferences секции
1. **Breeders hover**: детализация процентов использования (Divine Seeds 17%)
2. **Nutrients клик**: переход к обзорам удобрений (Boogie Brew 45%)
3. **Lights клик**: страница оборудования (Bestva Led 61%) 
4. **Compare**: сравнение предпочтений с другими гроверами

### F5: Content Sections
1. **Awards carousel**: листание наград/достижений
2. **Popular Diaries grid**: клик → переход к конкретному дневнику
3. **Grow Questions**: раскрытие thread решенных/открытых вопросов
4. **Activity Log scroll**: timeline активности с infinite scroll
5. **Reviews интеракция**: сортировка отзывов по сортам/рейтингам

### F6: Network секция
1. **Connected growers grid**: сетка связанных гроверов
2. **Mutual connections**: показ общих подписок
3. **Recommendations**: предложения новых подписок
4. **Social graph**: визуализация сети связей

### F7: Comments система
1. **Comment form**: создание нового комментария к профилю
2. **Thread navigation**: ответы на комментарии с nested структурой
3. **Reactions**: лайки/дизлайки комментариев
4. **Moderation**: отчеты о нарушениях, блокировка

### F8: Mobile адаптация
1. **Avatar touch**: touch-friendly увеличение
2. **Swipe tabs**: горизонтальный swipe между табами
3. **Responsive stats**: адаптация stats grid под узкие экраны
4. **Touch targets**: увеличение кнопок для touch интерфейса
