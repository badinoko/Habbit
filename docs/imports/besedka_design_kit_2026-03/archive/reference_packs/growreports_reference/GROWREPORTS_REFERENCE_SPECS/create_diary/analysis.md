# 🌱 Анализ Мастера Создания Нового Репорта GrowDiaries

**Дата анализа:** 01 сентября 2025  
**URL:** https://growdiaries.com/diaries/edit/new  
**Статус:** Полная документация страницы создания дневника

---

## 📋 СТРУКТУРА СТРАНИЦЫ СОЗДАНИЯ РЕПОРТА

### 🔗 1. Навигация и доступ
- **Кнопка доступа:** "Начать Мой Репорт" в правом верхнем углу
- **Расположение:** Рядом с кнопками "Логин" и "Регистрация"
- **URL:** `/diaries/edit/new`
- **Заголовок страницы:** "Новый Репорт"

### 👤 2. Секция регистрации (для новых пользователей)
```
Регистрация
├── Аватар пользователя (загрузка изображения)
├── Имя пользователя (текстовое поле)
├── Почта (текстовое поле)
├── Пароль (поле с показом/скрытием)
└── Повторите пароль (поле с показом/скрытием)
```

### 📝 3. Основная информация о репорте
```
Название Репорта
└── Текстовое поле для ввода названия
```

### 🌿 4. Секция сортов
```
Сорта
├── Описание: "Можно добавить до 8 сортов"
└── Кнопка: "+ Добавить Сорт"
```

### 🏠 5. Тип помещения (обязательный выбор)
```
Тип помещения
├── [ВЫБРАНО] В помещении (indoor)
├── Аутдор (outdoor) 
└── Greenhouse (теплица)
```

### 🏕️ 6. Оборудование для выращивания

#### **Grow Tents (Гроу-боксы)**
```
Grow Tents
└── Кнопка: "+ Add Tent"
```

#### **Освещение**
```
Лампы для веги
└── Кнопка: "+ Add Lamp"

Лампы для цвета

```

#### **Вентиляция и климат**
```
Ventilation Fans
└── Кнопка: "+ Add Ventilation Fan"

Air Filters
└── Кнопка: "+ Add Air Filter"

Air Conditioners

```

#### **Система полива**
```
Watering Type

├── Drip (капельный)

└── Aeroponics (аэропоника)

#### **Дополнительное оборудование**
```
Controllers

Substrates Used 0%
└── Кнопка: "+ Add Substrates"

CO2 generators

### 📊 7. Индикатор прогресса заполнения
```
Diary Completion Progress
├── Название Репорта (статус заполнения)
├── Сорта (статус заполнения)
└── Оборудование (статус заполнения)
```

### 💾 8. Сохранение
```
Сохранение
├── Предупреждение: "To save your Diary fill in all required fields"
└── Кнопка: "Сохранить" (кнопка сохранения черновика)
```

## 🎨 ВИЗУАЛЬНЫЕ ОСОБЕННОСТИ

### **Стиль кнопок добавления:**
- Цвет: Зеленый с иконкой "+"
- Текст: "+ Add [Название компонента]"
- Стиль: Простая зеленая кнопка с иконкой

### **Секции выбора типа:**
- **Тип помещения:** Карточки с иконками и названиями
- **Watering Type:** Аналогичные карточки с визуальными иконками
- **Выбранные элементы:** Подсвечены зеленой галочкой

### **Прогресс заполнения:**
- Отображается внизу страницы
- Показывает статус каждой обязательной секции
- Индикатор прогресса в процентах для некоторых секций

---

## 🔧 ФУНКЦИОНАЛЬНЫЕ ОСОБЕННОСТИ

### **1. Модульность:**
- Каждый тип оборудования добавляется отдельно
- Возможность добавления множественных элементов одного типа

### **2. Валидация:**
- Обязательные поля: Название репорта, Сорта, Оборудование
- Кнопка сохранения активна только при заполнении всех обязательных полей

### **3. Копирование настроек:**
- Возможность скопировать лампы из секции "Веги" в секцию "Цвета"

### **4. Ограничения:**
- Максимум 8 сортов на один репорт
- Процентный индикатор использования субстратов
