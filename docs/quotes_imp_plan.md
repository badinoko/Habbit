# Шаг 7. Реализовать сервис refill

## Цель
Основная бизнес логика загрузки батча.

## Что сделать

Создать `refill_service.py`.

Метод:

```

refill_quotes()

```

Алгоритм:

1. создать новый batch (`pending`)
2. получить 50 цитат
3. перевести их
4. сохранить переводы
5. проверить количество успешных переводов
6. если >= порога:
   - деактивировать старый batch
   - активировать новый batch
7. иначе:
   - пометить batch как `failed`

## Критерии готовности

- новый batch активируется только после успешной загрузки
- старый batch не ломается при ошибке
